# Minecraft DoMCer服务器玩家数据排行榜

## DoMCer服务器是什么：

一个基于《我的世界》二次开发并运营的服务器团队，目前游戏已上架 网易《我的世界》平台 我们致力打造一个与众不同的游戏世界，团队目前已经有4年以上的时间沉淀

----



服务器内容包括：

- 起床战争（BedWars）

- 空岛战争（SkyWars）

- 超级战墙（UHC）

- 战桥（Bridge）

- 记忆速建

- 趣味生存 等
  
  

---

## 网站构成：

- 玩家数据查询

- 公会数据查询

- 玩家战绩查询

- 排行榜

- 个人中心

---

## 网站设计初衷：

DoMCer服务器是网易吾世界公司旗下游戏我的世界中一个非常有魅力的服务器，因其无私的开源精神和特色的游玩内容受到了广大玩家的喜爱，但是因为各种各样的原因包括但不限于：国内Minecraft社区缩水下滑严重、网易自身管理欠妥、DoMCer服务器管理外挂手段不彻底等，导致DoMCer在2023年游玩人数骤降，全服平均日活从800降至100。本网站的主要功能是针对本服务器数据进行呈现。方便玩家查询自己在服务器的各项数据，在此非常感谢DoMCer对广大第二开发者公开的数据API。
